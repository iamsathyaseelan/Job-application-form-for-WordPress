Collect job applications for your company with a simple plugin.

== Description ==

Show the job application collection form on any of your pages using the shortcode "[applicant_form]".

* No need for configuration.
* Send emails to the applicant once the application is submitted.
* You can view the latest applications in your dashboard.
* You can search for applications based on the applicant's name, email and post name.
* You have full control of arranging the applications using the submitted date.

Upcoming features:

1. Customize the application form.
1. Send and receive emails inside your site.


== Installation ==

It is very easy to install our plugin.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Paste the shortcode [applicant_form] on any pages you want.
